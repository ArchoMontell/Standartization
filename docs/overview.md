# 1. Опис системи

## Система
REST API сервіс для бронювання квитків.

Система призначена для автоматизації процесу пошуку рейсів, створення бронювання, оплати та керування замовленнями. Взаємодія здійснюється через REST API.

Користувачі можуть:
- шукати рейси;
- бронювати квитки;
- оплачувати замовлення;
- переглядати історію.

Адміністратори:
- керують рейсами;
- контролюють доступність місць.

## Формат взаємодії
REST API (backend сервіс).
