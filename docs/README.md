## Принципи Documentation as Code

- Уся документація зберігається у форматі Markdown.
- Документація версіонується разом із проєктом у Git.
- Документи мають логічну структуру та пов’язані між собою посиланнями.
- Для публікації використовується генерація статичного сайту.
